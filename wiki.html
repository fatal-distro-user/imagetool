Image Format Wiki

Learn about different image formats and their uses.

JPEG (Joint Photographic Experts Group)

JPEG is a commonly used format for digital images, especially for photographs. It uses lossy compression, which reduces file size significantly but may result in a slight loss of quality. Ideal for web use and sharing.

Learn more about JPEG

PNG (Portable Network Graphics)

PNG is a lossless image format that supports transparency. It is widely used for graphics, logos, and images requiring high quality and transparency.

Learn more about PNG

GIF (Graphics Interchange Format)

GIF supports animation and is commonly used for short, looping animations. It uses a limited color palette, making it less suitable for detailed images.

Learn more about GIF

WEBP

WEBP is a modern image format that provides superior compression for images on the web. It supports both lossy and lossless compression, as well as transparency and animation.

Learn more about WEBP

BMP (Bitmap)

BMP is an uncompressed image format that retains high quality. However, its large file size makes it less practical for web use.

Learn more about BMP

TIFF (Tagged Image File Format)

TIFF is a versatile format often used in professional photography and publishing. It supports lossless compression and high-quality images.

Learn more about TIFF

SVG (Scalable Vector Graphics)

SVG is a vector image format ideal for graphics that need to scale without losing quality. It is widely used for icons, logos, and illustrations.

Learn more about SVG

RAW

RAW files are unprocessed images captured directly from a camera sensor. They retain all image data, making them ideal for professional editing.

Learn more about RAW

HEIF (High Efficiency Image Format)

HEIF is a modern format that provides high-quality images with smaller file sizes. It is commonly used on Apple devices.

Learn more about HEIF

ICO (Icon)

ICO is a format used for icons in software and websites. It supports multiple sizes and resolutions within a single file.

Learn more about ICO
